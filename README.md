# Nick's Daily Workout

Publiceer via GitHub Pages:

1. Maak een nieuwe repository met de naam `workout`.
2. Upload alle bestanden uit deze map naar de hoofdfolder van de repository.
3. Open **Settings → Pages**.
4. Kies bij **Build and deployment**: **Deploy from a branch**.
5. Selecteer branch **main** en folder **/(root)**.
6. Klik **Save**.

De app verschijnt daarna op:

https://nickmorsch.github.io/workout/
